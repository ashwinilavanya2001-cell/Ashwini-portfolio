My Projects

📊 Real-Time News Sentiment Analysis System

Real-Time News Sentiment Analysis is a web application that collects live news articles using NewsAPI and analyzes their sentiment using Natural Language Processing (NLP). It classifies each article as Positive, Negative, or Neutral and displays the results in an interactive Streamlit dashboard, helping users quickly understand public opinion on current news.

📧Privacy Preserving Multi-Client Spam Detection System

This project detects spam messages while protecting user privacy using Federated Learning and Agentic AI. It uses MiniLM, BERT, and XGBoost to accurately classify messages as spam or non-spam. The system continuously monitors its performance and automatically retrains the model when needed, improving accuracy without sharing sensitive user data.

📧Agentic AI-Based Crime Anomaly Detection and Predictive Analytics System

This project analyzes crime data using Machine Learning and Agentic AI to detect unusual crime patterns and predict future crime trends. It identifies crime hotspots, provides data visualizations, and uses SHAP (Explainable AI) to explain the model's predictions, helping authorities make informed decisions for crime prevention.
